# Visual Question Answering via Multimodal Traceability Mesh (MTM)

**A Provenance-Aware Framework for Grounded and Verifiable Document Reasoning**

## 📖 Overview
Large Vision Language Models (LVLMs) struggle with "source hallucination"—confidently providing answers not backed by real evidence. **Multimodal Traceability Mesh (MTM)** is a framework that ensures every piece of information generated by the AI is strictly anchored to specific coordinates (FileID, Page, and Bounding Box) within the source document. By treating provenance as a strict rule rather than a "soft suggestion," MTM triggers a **hallucination breakpoint** if a claim cannot be verified, choosing silence over confabulation.

---

## 🚀 Key Architectural Contributions

*   **Neo4j Provenance Graph:** A labeled property graph that maps document structure (Document → Page → Block → Table/Image) with immutable coordinate triads.
*   **Hybrid Graph-Vector Retrieval:** Combines **CLIP-ViT-L/14** semantic vector search with multi-hop Cypher traversal to surface context that is both semantically relevant and structurally coherent.
*   **CitationLogitMasker:** A novel decoding-time constraint that intercepts raw logit scores to prevent the model from generating node identifiers outside the retrieved provenance vocabulary.
*   **Dual-Validator:** A five-stage validation pipeline (Citation Presence, BBox Presence, Graph Existence, IoU Check, Coordinate Validity) that enforces a hard breakpoint on ungrounded claims.
*   **Visual Mesh Synthetic Overlay:** A modality-bridging technique that renders graph nodes and relationships directly onto document images for spatial grounding.

## 🛠 Tech Stack

*   **Model Backbone:** Llama-3.2-11B-Vision-Instruct (Fine-tuned with QLoRA: $r=16, \alpha=16$)
*   **Knowledge Graph:** Neo4j 5.15
*   **Vector Database:** FAISS (via CLIP-ViT-L/14)
*   **OCR & Layout:** PaddleOCR, PyMuPDF, pdfplumber
*   **Deployment:** Flask REST API with Audit Logging for all validation decisions.

## 📋 Quick Start

### 1. Installation
```bash
git clone https://github.com/your-org/multimodal-traceability-mesh.git
cd multimodal-traceability-mesh
pip install -r requirements.txt
```

### 2. Configuration
Setup your `.env` for Neo4j and HuggingFace:
```env
NEO4J_URI=bolt://localhost:7687
NEO4J_USER=neo4j
NEO4J_PASSWORD=your_password
HF_TOKEN=your_huggingface_token
```

### 3. Execution
Run the comprehensive evaluation suite:
```bash
bash run_total_eval_slurm.sh
```
